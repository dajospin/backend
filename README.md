Mekanbul Backend
Demo: [https://mekanbul-backend-bice.vercel.app/](https://backend-chi-flame-39.vercel.app/)

Bu repo mekanbul-backend uygulamasının basit Node.js + Express + Mongoose backend'dir.

Kısa Açıklama:
API, mekan (venue) verilerini yönetir: listeleme, ekleme, görüntüleme, güncelleme ve silme.

MongoDB Cloud kullanır. Bağlantı bilgisi: (bak: app_api/models/db.js).

Kurulum

cd /path/to/backend
npm install

Uygulamayı çalıştırma
npm start
